# Keystone School Management System

A School Management System built using **KeystoneJS 6**, **GraphQL**, and **TypeScript**.  
This project allows managing Students, Teachers, Departments, Subjects, Marks, and Attendance records efficiently through a simple Admin UI and flexible GraphQL API.

---

## Tech Stack
- Backend Framework: [KeystoneJS 6](https://keystonejs.com/)
- Language: TypeScript
- Database: SQLite (can be switched to PostgreSQL or MongoDB)
- API: GraphQL
- Authentication: Keystone Auth
- Storage: Local image upload (student profile images)

---

## Features
- Manage Students, Teachers, Departments, Subjects, Marks, and Attendance
- Role-based data relationships using Keystone schema
- GraphQL API for querying and filtering data
- Admin Dashboard for CRUD operations
- Type-safe backend written in TypeScript
- Easily extendable for React or Next.js frontend integration
